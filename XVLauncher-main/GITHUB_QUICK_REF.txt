🔧 KONFIGURATION (Einmalig)
───────────────────────────────────────────────────────────────
1. GitHub Token erstellen:
   https://github.com/settings/tokens
   → Scope: "repo" oder "public_repo"
   
2. Resources.resx bearbeiten:
   AccessToken: ghp_DEIN_TOKEN
   UpdateUrl: https://raw.githubusercontent.com/USERNAME/REPO/{0}/
   
3. Settings.settings bearbeiten:
   ProjectID: USERNAME/REPONAME
   GameDirectory: Game

🐛 TROUBLESHOOTING
───────────────────────────────────────────────────────────────
Problem: "401 Unauthorized"
→ Token in Resources.resx prüfen

Problem: "404 Not Found"  
→ ProjectID in Settings.settings: USERNAME/REPONAME

Problem: "Could not find release"
→ Mindestens 1 GitHub Release erstellen

Problem: Launcher zeigt "DOWNLOAD" statt "SPIELEN"
→ GameDirectory = "Game" in Settings.settings

Problem: Keine Graphics im Spiel
→ Game.exe muss der Launcher sein (106 KB)

🎮 USER EXPERIENCE

2. Startet Launcher

4. Nur geänderte Dateien werden geladen
